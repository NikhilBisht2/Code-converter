# Code Converter: C ↔ Python
This is a simple web application that converts code between **C** and **Python** using an AI model. Users can input C or Python code, select the desired conversion direction, and instantly receive the converted code.

# Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js + Express + Openrouter API
- **AI Model:** deepseek/deepseek-r1-0528-qwen3-8b:free
